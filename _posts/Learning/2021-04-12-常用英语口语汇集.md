### 词性转换七大核心规则


<img src="/public/zimage/english/word_change.jpg"  width="100%" height="100%"/>
####  一、动词 → 名词
```



一、动词 → 名词
规则：添加后缀 -tion/-sion、-ment、-ance/-ence 等。
示例：

solve → solution（解决方法）
achieve → achievement（成就）
perform → performance（演出）
advertise → advertisement（广告）
commit → commitment（奉献）



```

####  二、动词 → 名词/形容词规则：

```
二、动词 → 名词/形容词规则：

名词：添加 -or/-er（指人）、-ion（动作/结果）。
形容词：添加 -ive、-able。
示例：
act → actor（演员） / action（行动） / active（积极的）
change → change（名词：变化） / changeable（可变的）
```


####   三、动词 → 形容词/副词/名词

```
三、动词 → 形容词/副词/名词
规则：

形容词：-ing（令人…）、-ed（感到…）、-ful、-ous。
副词：形容词 + -ly。
名词：-tion、-ness。
示例：
interest → interesting（有趣的） / interested（感兴趣的）
fill → full（满的）
need → necessary（必要的） / necessarily（必要地）
```



#### 四、名词 → 形容词

```
四、名词 → 形容词
规则：添加后缀 -al、-ous、-ful、-y、-less。
示例：

person → personal（个人的）
danger → dangerous（危险的）
care → careful（小心的） / careless（粗心的）
sun → sunny（晴朗的）
fun → funny（有趣的）
```



####  五、名词 → 复数名词
```
五、名词 → 复数名词
规则：

规则变化：加 -s/-es（如 book → books）。
不规则变化：
man → men
child → children
gentleman → gentlemen

```





#### 六、形容词 → 副词

```
六、形容词 → 副词
规则：

直接加 -ly（如 quick → quickly）。
以 -y 结尾：变 y 为 i 加 -ly（如 happy → happily）。
特殊变化：
true → truly（去 e 加 -ly）
good → well
```


#### 七、形容词 → 副词/名词

```
七、形容词 → 副词/名词
规则：

副词：形容词 + -ly（如 possible → possibly）。
名词：添加 -ness、-ity、-th。
示例：
lucky → luckily（副词） / luck（名词）
true → truly（副词） / truth（名词）
able → ability（能力）
warm → warmth（温暖）

```
