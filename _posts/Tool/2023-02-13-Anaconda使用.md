
## conda 重命名环境

conda 没有提供重命名环境的命令，我们只能先克隆一份原来的环境，然后再删除原来的环境，
具体操作如下（例如将环境 a_env 重命名成 b_env ）：

```
conda create -n b_env --clone a_env
conda remove -n a_env --all


```
